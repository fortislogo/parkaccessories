A) Overview - top

Clean and modern theme design allows you to use this theme for every kind of online shop.

B) Theme installation - top

  1. If you are using Windows based (XAMP/MAMP/Other) hosting/server be shure that PHP settings "short_open_tag" is set to ON. Contact your hosting probider if you dont know how to set this PHP value, before installing theme. If your short_open_tag = OFF you will get error "unexpecting end of file" after theme installation on Windows based server.
  2. First of all install latest official Open Cart version from Open Cart website.
  3. Open folder UPLOAD 1.5.5.1 or UPLOAD 1.5.4.X depending on what OpenCart version you are using from purchased archive and upload all content inside it (folders image, catalog and admin) to your installed Open Cart directory:

    You can Drag and drop the admin, image and catalog folders into your site using your favourite FTP client (If you don’t have one, we recommend Filezilla).

    You will be asked if you want to replace some of the files from your current installation - you must click YES to replace all files.
    Don't worry - no core files will be replaced, all your old modules and themes will works as always.

    If you are Mac OS user we strongly recomend to use FileZilla client for theme upload, becouse MacOS finder can replace folders with files incorrect!
  4. Go to your open cart directory /image/data in FTP client, and setup write premissions (chmod 666) for file theme_colors.bst:

  5. Go to your Open Cart admin panel System > Settings > Edit. Click Store tab and choose beatyshop theme for "Template" option. This action will activate your new theme.
  6. Go to your Open Cart admin panel System > Settings > Edit. Click Image tab to open Image settings.
    Setup this image sizes:
  7. IMPORTANT: If you want that your product images and thumbnails will fit theme nice (like on demo site) we recomend your to use 1000x1085 px image size for your product photos! (or other size proportional to this size).
  8. Go to your Open Cart admin panel System > Settings > Edit. Click Option tab and set Default item per page value to 9:

  9. In your Open Cart admin go to Extensions > Modules.
    In ANY module with product display (Features, Bestsellers, Latest, Special, etc) that you want to use now or later you must set correct image size (188x204) for product display:

  10. In your Open Cart admin go to Extensions > Modules and click Install near to "BeautyShop Theme Administration" moduie:
  11. In your Open Cart admin go to Extensions > Modules and click Edit near to "BeautyShop Theme Administration" moduie. You will open theme control panel.
  12. Choose "Enable" for Custom theme options:
  13. Click "Save settings theme settings" button first time, becouse you use this module firstly.
    Now module is setuped to work with Theme, but you need to configure some features that you need in this theme admin panel, configurate and enable custom footer, etc.

C) Slider setup - top

  1. Go to System > Design > Banners and Click Insert button to make new Images group for slider. We name group "Slider", you can use name what you want.
  2. Click Add banner and upload your slider image and provide link and title for it.
    IMPORTANT: Does not use words "ads" and "mini" in your slider images title field!

    Recommended slider image sizes:
    If you will use layout with mini slider images at the right - recommeded slider image size is 690x480 px
    If you will use layout with big full page width slider images - recommeded slider image width is 980 px

  3. Repeat step 2 for your every slider image (you can use sample "slider" images from "SAMPLE SLIDER IMAGES" directory).
  4. You can add Mini Sliders and/or Slider ads images to your slider:
  5. If you want to show two (or more) mini sliders at the right of slider (like on demo page):
    1. Add mini slider image with size 282x241px as described in 2 step, but input title with "mini" word inside it (for example title can be "My mini slider 1". This is necessarily for engine to know that this is Mini slider image for right display:
    2. Add TWO mini sliders images and Save (you can use sample "minislider" images from "SAMPLE SLIDER IMAGES" directory).

  6. If you want to show three (or more) mini ads images at the bottom of slider (like on demo page):
    1. Add ads slider image with width 314px or smaller and any height as described in 2 step, but input title with "ads" word inside it (for example title can be "ads 1". This is necessarily for engine to know that this is Ads slider image for bottom display:
    2. Add three mini ads images and Save (you can use sample "ads" images from "SAMPLE SLIDER IMAGES" directory).

    You can add more than two mini sliders and more than three ads images, but you must calculate right image sizes corresponding to your main slider image size if your want to theme look nice.

  7. Go to Extensions > Modules, click Edit for Slideshow module and make sure your are enabled slideshow module on Home page (or other pages). Change Banner item to your Banners group name that your created in 1 step:

D) Manage theme layouts (1,2 or 3 columns) - top

This theme support different content layouts, you can use one, two or three columns in your theme, and use its different combinations in your store pages.

If you want to use 1 column layout on some (or all) shop pages:


(1 column layout example)

  1. Go to Extensions > Modules, click Edit near EVERY module that you use and DISABLE all modules from displaying in Content Left and Content Right positions. You can show your modules in Content top and Content Bottom positions only if you want to get 1 column layout for your theme:

If you want to use 2 column layout on some (or all) shop pages:


(2 column layout example)

  1. Go to Extensions > Modules, click Edit near EVERY module that you use and enable some modules for Content Left position OR for Content Right position. You can show your modules in Content top and Content Bottom positions too.

If you want to use 3 column layout on some (or all) shop pages:


(3 column layout example)

  1. Go to Extensions > Modules, click Edit near EVERY module that you use and enable some modules for Content Left position AND some modules for Content Right position. You can show your modules in Content top and Content Bottom positions too.

E) Welcome module setup - top

You can show welcome text and special images/banners on your pages or home page:

To enable this feature:

  1. In your Open Cart admin go to Extensions > Modules and click Install near to "Welcome" moduie.
  2. Click Edit near Welcome module.
  3. Set Layout value to Home, position Content Top, adjust sort order with your other modules.
  4. Click Source button in Welcome Message Text Editor.
  5. If you want to show banners like on demo copy and paste this html code to editor:

    <div class="welcome-buttons">
    <a href="paste-you-link-here"><img class="fade-image" src="catalog/view/theme/beautyshop/image/welcome1.png" /></a><a href="paste-you-link-here"><img class="fade-image" src="catalog/view/theme/beautyshop/image/welcome2.png" /></a></div>

  6. You can change "paste-you-link-here" to your image link if your wish to make a link to images. You can add your text or images too.
  7. Save module.

F) Manage color themes - top

In powerful theme control panel your can manage color themes, change all colors for theme elements and more!

  1. In your Open Cart admin go to Extensions > Modules and click Edit near to "BeautyShop Theme Administration" moduie. You will open theme control panel.
  2. Color tab is opened by default.
  3. To change theme color for some element you can click inside element color field and use appeared color picker to change color.
  4. To preview your color click "Save settings theme settings" button and refresh your website page (open your site in another tab).
  5. If you dont like your color and want to revert it back you can select "Beauty default" theme near to "Load color scheme" button and click this button to load default color. After that click "Save settings theme settings" to revert back colors on your site.
  6. You can create your own color themes - after you choose all colors that you want to change input Theme Title instead of "My theme" text and click "Save color scheme" button. Now you can find you color theme inside Load scheme list and loat it if your want.
  7. To delete your color scheme select it from drop down list and click Delete Selected Scheme button.

G) Theme administration panel usage - top

  1. In your Open Cart admin go to Extensions > Modules and click Edit near to "BeautyShop Theme Administration" moduie. You will open theme control panel.
  2. Use left tabs to access all theme features settings and setup what you want.
  3. Dont forget to click "Save settings theme settings" to apply your changes!
